Array

Não precisamos definir um tipo para o array, ele pode ter variáveis de diversos tipos

Função anônima

É executada automaticamente, não precisa ser chamada
function(){}
OU como arrow function:
(elemento)=>{códgio}



Códigos

Array
var array = new Array(3)
array[0] = 1
console.log(array)

Método lenght
Retorna o tamanho do array:
array.lenght

Método push
Adiciona um elemento no final do array
array.push("teste")

Método unshift
Adiciona um elemento no início do array
array.unshift("unshift")

Método delete
Remove um valor determinado, deixando o espaço vazio (empty)
delete array[1]

Método pop
Remove o último elemento e também a sua posição, retorna o valor do item removido
var val = array.pop()

Método shift
Remove o primeiro elemento e também a sua posição, retorna o valor do item removido
var val = array.shift()

Método splice
Remove os valores determinados, tanto seu valor quanto sua posição.
array.splice(1,1)
1º argumento -> A partir de qual elemento vai começar a exluir
2º argumento -> Define quantos valores vai apagar a patir do index determinado

Método concatenar
Concatena arrays
a1.concat(a2)

Método join
Printa um vetor e a cada valor adiciona o elemento indicado
a1.join("/")

Método slice
Pega uma parte de um vetor, com base nos indíces
a1.slice(0,1)

Método reverse
Inverte a ordem do vetor
a1.reverse()

Método sort
Ordena o vetor de forma crescente
instrumentos.sort()

Método map
Mapeia o vetor e faz a ação declarada na função anônima
lista.map(function(item) {return item* 5})
lista.map((item)=>{return item*5})

Método reduce
"Reduz o vetor", e realiza alguma ação com os valores do vetor.
total=lista.reduce((acumulador,item)=>{return acumulador+item;})
1ºArgumento -> Acumulador que ficará fixo rodando o vetor
2ºArgumento -> O item atual no ato de percorrer o vetor

Método findIndex
Retorna o index em que aquele elemento especificado está
index = vetor.findIndex((elemento)=>{return elemento==maçã})

Método find
Retorna o valor do elemento requirido
valor = vetor.find(x=>x>45)

Método includes
Retorna true ou false, dependendo se aquele elemento está no vetor ou não
nomeTeste.includes('Simone');

Método some
Retorna true ou false, dependendo se aquela condição está presente em algum elemento do vetor
nomeTeste.some(nome=>nome=='Lucas');

Método every
Retorna true ou false, dependendo se aquela condição está presente em todo o vetor
const maiorQue3 = numeros.every(x=>x>3);

Método filter
Retorna um vetor com os elementos do vetor princiapla que respeitam a condição
buscarMaior45 = numeros.filter(x=> x>3);

Método forEach
Percorre o vetor como um for mas como método
carros.forEach(function(item,index,array){array[index] = 'Carro '+item})

Spread operator
Concatena vetores
let vfinal = [...v1,...v2];

Destructive ...
Forma diferente de atribuir um elemento do vetor a uma variável
Forma padrão: let variavelResultante = dados[1];
2.1: let {index:nomeVariávelResultandte} = vetorInicial;
2.2: let {index:nomeVariávelResultandte, index2:nomeVariávelResultandte2} = vetorInicial;
3: let[primeiroValor,segundoValor,ultimoValor]=vetorInicial; -> Assim não precisa determinar o index entrentanto a quantidade de valores deve ser a mesma do vetor inicial
