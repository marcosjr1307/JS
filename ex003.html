Array

Não precisamos definir um tipo para o array, ele pode ter variáveis de diversos tipos



Códigos

Array
var array = new Array(3)
array[0] = 1
console.log(array)

Método lenght
Retorna o tamanho do array:
array.lenght

Método push
Adiciona um elemento no final do array
array.push("teste")

Método unshift
Adiciona um elemento no início do array
array.unshift("unshift")

Método delete
Remove um valor determinado, deixando o espaço vazio (empty)
delete array[1]

Método pop
Remove o último elemento e também a sua posição, retorna o valor do item removido
var val = array.pop()

Método shift
Remove o primeiro elemento e também a sua posição, retorna o valor do item removido
var val = array.shift()

Método splice
Remove os valores determinados, tanto seu valor quanto sua posição.
array.splice(1,1)
1º argumento -> A partir de qual elemento vai começar a exluir
2º argumento -> Define quantos valores vai apagar a patir do index determinado

Método concatenar
Concatena arrays
a1.concat(a2)

Método join
Printa um vetor e a cada valor adiciona o elemento indicado
a1.join("/")

Método slice
Pega uma parte de um vetor, com base nos indíces
a1.slice(0,1)

Método reverse
Inverte a ordem do vetor
a1.reverse()
