Data atual
Formas de instanciar um tipo date
var data = new Date(); -> Data atual
var dataPassada = new Date("01/22/1994 15:23:35"); -> Data passada
var data2 = new Date(1994,0,22,15,23,35); -> Data passada

Métodos da clase date
Pega a data
var dia = hoje.getDate(); -> Pega o dia
var mes = hoje.getMonth(); -> Pega o mês, sendo que janeiro é 0
var ano = hoje.getFullYear(); -> Pega o ano
var diaSemana = hoje.getDay(); -> Pega o dia da semana, sendo que domingo é 0

Tratamento de Date
Impossível usar o comparador 'igual' (==), pois ao instanciar um tipo date ele pega os valores dos milisegundos também, ou seja, essa comparação sempre será falsa
Uma solução é usar o método getTime()
Já realizar comparações como maior e menor são válidas
if(dataX == dataY) -> Errado
if(dataX.getTime() == dataY.getTime()) -> Correto
if(dataX > dataY) -> Correto
